# Changelog

所有对该项目的重要更改都将记录在此文件中。

## [未发布]

### 重构
- **弹幕渲染系统重大重构**：从自研渲染器迁移到模块化组件库架构
- 将弹幕渲染核心抽象为独立的 DanmakuRenderer 组件
- 实现了渲染逻辑与业务逻辑的完全分离
- 优化了组件间的通信机制，采用 props 和事件系统
- 提升了代码可维护性和扩展性

### 新增
- 组件化弹幕渲染引擎，支持更灵活的配置选项
- 弹幕交互事件系统，支持点击、可见性变化等事件监听
- 完善的组件参数文档和使用示例

### 改进
- 优化了弹幕渲染性能，提高了高并发场景下的稳定性
- 增强了跨浏览器兼容性，特别是针对滑块控件的样式处理
- 提升了系统整体架构的可维护性和可扩展性

### 修复
- 修复了部分浏览器中 CSS appearance 属性的兼容性问题
- 解决了组件卸载时的资源清理问题

## [1.0.0] - 2025-10-19

### 新增
- 完整的实时弹幕系统核心功能
- 前端弹幕渲染引擎，支持滚动、顶部固定、底部固定三种样式
- 防重叠渲染机制，智能分配弹幕位置
- WebSocket实时通信，基于Socket.io 4.x
- 多房间管理，支持万人级直播间
- Redis集群支持，实现横向扩展
- 敏感词过滤，使用Trie树算法
- 频率限制和反刷屏机制
- 可选的MongoDB历史消息存储
- 高并发压力测试脚本
- Docker Compose一键部署配置

### 改进
- 优化WebSocket连接稳定性
- 提升前端渲染性能
- 完善错误处理和日志记录

### 修复
- 修复部分边缘情况下的连接问题
- 修复弹幕渲染的兼容性问题
